フリー誘発タイミング
【効果の後誘発】（4回）
・自分・相手の効果発動後（戦場・魔罠・手札・墓地）
・自分・相手の戦闘宣言→戦闘効果の後・
・自分・相手の戦闘によるダメージを受けた後
・自分・相手の攻撃の終了後

【フェイズ切り替わり】（5回）
・スタンバイフェイズ開始時
・バトルフェイズ開始時
・バトル終了時（メイン２開始時）
・エンドフェイズフェイズ開始時
・ターンの切り替え前の解消

【モンスターの召喚誘発】（2回）
・EXからモンスターが出たとき/効果を使った時
・モンスターの召喚後
（特殊召喚はそもそも効果で統一されているので、その効果処理後にタイミングがある）



【終了処理一覧】：
効果後誘発のCHECK_FIN＿DATAで確認できるいろいろな情報だゾ
例として：自分_○○_保、相手_○○_保にデータを効果処理が終わるたびにロードして、
それらを効果後処理にぶち込むぞ。
―【BIT系統】―――――――――――――――――――――――――――――――――――――――――
・いろいろな、BITで保存できる情報の事。カードの出所とか
CALL CARD_CHECK_FIN_BIT(決闘者,"離移動元",3,2)
SIF RESULT != 1
	RETURN 0
エクソシスターの例：決闘者の設定、確認したい種類の設定、BITの設定、指定プレイヤー設定(0or未入力なら設定決闘者、1なら相手のみ、2ならどちらも)
これで、お互いの墓地からカードが移動した場合、正を返す関数となる。

-離移動元
・カードがどこから離れたのかを確認するBIT変数、下記該当BIT
　戦場 = 0,魔罠 = 1,手札 = 2, 墓地 = 3,デッキ = 4,EXデッキ = 5, 除外 = 6, トークン生産 = 7

-効発動元
・カードがどこで発動したかを確認するBIT変数、下記該当BIT
　戦場 = 0,魔罠 = 1,手札 = 2, 墓地 = 3,デッキ = 4,EXデッキ = 5, 除外 = 6, トークン生産 = 7

-召喚出処
・カードがどこから召喚・特殊召喚したか確認するBIT変数、下記該当BIT
　戦場 = 0,魔罠 = 1,手札 = 2, 墓地 = 3,デッキ = 4,EXデッキ = 5, 除外 = 6, トークン生産 = 7

-被弾種
・受けたダメージの種類を確認するBIT変数、下記該当BIT
　戦闘 = 1,直接戦闘 = 2, 効果 = 3

-ダイス目
・出したダイスの出目を保存したBIT変数。該当BITは1～6

―【CT付き系統】】―――――――――――――――――――――――――――――――――――――――――
・カードの情報などを保存する領域。この変数には効果条件を使用でき、そこからカードIDを確認できる。

ドーハスーラの例：決闘者の設定、確認したい種類の設定、指定プレイヤー設定(0or未入力なら設定決闘者、1なら相手のみ、2ならどちらも)
CALL FULLRESET
効果条件固名前 = 死霊王 ドーハスーラ
効果条件種族 = アンデット族
CALL CARD_CHECK_FIN_DATA(決闘者,"効果発動",2)
SIF RESULT != 1
	RETURN 0
これで、《死霊王 ドーハスーラ》が以外のアンデット族モンスターが効果を発動した場合、正を返す関数となる。

ドラゴンメイドの集団ご奉仕
通常魔法
このカード名の②の効果は１回のデュエルにつき一度までしか発動できない。
①：自分フィールドに「ドラゴンメイド」モンスターが3体以上いるときにこのカードを発動できる。
墓地の「ドラゴンメイド」カードをフィールドの「ドラゴンメイド」モンスターの数までデッキに戻す。
このカードの発動後このカードを手札に戻す。
②：このカードをドローした場合、このカードを相手に見せ、以下の効果を適用する：
●自分はこのデュエル中、「ドラゴンメイド」モンスターしか召喚できない。
●自分はドラゴンメイドモンスターの召喚に際し、召喚権を消費しない
＊この効果を発動した後、このカードの所有者は今日一日デュエル外で「ドラゴンメイド」以外のカードを召還できない。

＞所感：
システム的に言うと、全然再現は容易……。①の効果にかみ合いを感じられないかも。

これでしたら①の条件に②の効果を適用したり、残りの2体をデッキ/墓地から特殊召喚する効果がいいかも。

実際のデュエルとの相違点：

・選ぶは選択に統一されており、対象耐性の影響を受ける
・手札で発動した通常/速攻魔法や罠カードなどは手札から直接墓地に行く。
・リバースは独自の効果が多すぎるため廃止している
・裏側除外がない
・蘇生制限がない

・フリーチェーンは基本相手の効果に反応して発動できない（チェーン1でしか発動できない）
・攻撃時や、効果の後に発動できる効果は1つまで(例えばミラフォが２つある場合、一つ発動からもう一つ発動とかはできない)
　強制効果は一応複数回連なる
・基本的に除去系の効果で自分のカードを選べないようにしている（CPUが自分のカードを無意味に消すのを抑制）
・から打ちできるカードはいくらか存在している。（デッキからカードを加える効果など）
・作成されたトークンはカードとして存在している
・いくらかの効果は共通化されて、キーワードにされている。（戦闘破壊耐性など）
・墓地のカードに情報を付与することはできない（ハ・デス系の効果）
・フィールドのカードに効果付与作成は可能だが、開発が厳しく、開発者の負担が非常に大きい
・カードを対象にしたとき起動する効果などはバグの苗床すぎるので作成不可

・リンクに守備表示がある、リンク先にしか出せない制限はない、レベルはあるがS素材としては使用できない
・エクシーズにもレベルがあるがS素材には使用できない
・X素材にされたモンスターは墓地に送られ、
その分のX素材というカウンタが乗るだけ。なのでカードをX素材にすることはできない。
